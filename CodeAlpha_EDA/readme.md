## 🚀 Key EDA Stages

* **Data Inspection:** Assessing the structure, dimensions, and data types of the dataset.
* **Data Cleaning:** Handling missing values, removing duplicates, and fixing inconsistent data formats to ensure analysis reliability.
* **Univariate Analysis:** Analyzing individual variables (e.g., distribution of age, salary, or sales) using histograms and box plots.
* **Bivariate/Multivariate Analysis:** Exploring relationships between two or more variables using scatter plots, correlation matrices, and heatmaps.
* **Anomaly Detection:** Identifying outliers that might skew results or provide unique insights.
